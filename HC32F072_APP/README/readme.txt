【功能描述】
================================================================================
本程序实现接收CAN数据，根据协议进行解析，发出UART数据。

================================================================================
【测试用板】
================================================================================
俄罗斯协议转接板(HC32F072KATA)

================================================================================
【测试工具】
================================================================================
USBto232转接板
USBtoCAN转接板

================================================================================
【测试软件】
================================================================================
SSCOM5
USB-CAN-TOOL

================================================================================
【使用步骤】
================================================================================
1、重新编译代码运行
2、串口调试工具配置波特率9600bps、偶校验、1stop
3、CAN调试根据配置波特率1000Kbps

================================================================================
【参数说明】
================================================================================
[UART1]
PA02: Tx
PA03: Rx
波特率: 9600bps

[CAN]
PB08: RX
PB09: TX
波特率: 1000Kbps
